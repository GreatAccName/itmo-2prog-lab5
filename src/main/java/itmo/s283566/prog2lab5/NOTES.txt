help : вывести справку по доступным командам.
info : вывести информацию о коллекции квартир.
show : вывести все квартиры коллекции в строковом представлении.
clear : очистить коллекцию.
save : сохранить коллекцию в файл.
exit : завершить программу (без сохранения в файл).
print_descending : вывести квартиры коллекции в порядке убывания.
min_by_id : вывести квартиру с минимальным ID.
remove_lower [AREA] : удалить из коллекции все квартиры, площадь которых меньше заданной AREA.
update [ID] : обновить данные квартиры коллекции по ID (далее ввод данных квартиры).
execute_script [FILE_NAME] : исполнить скрипт из указанного файла (можно не указывать).
insert [KEY] : добавить новую квартиру с заданным ключом (далее ввод данных квартиры).
remove_key [KEY] : удалить квартиру из коллекции по его ключу KEY.
filter_starts_with_name [SUBNAME] : вывести квартиры, значение имени которых начинается с заданной подстроки SUBNAME.
replace_if_greater [KEY] [AREA] : заменить значение площади квартиры по ключу, если новое значение больше старого.
replace_if_lower [KEY] [AREA] : заменить значение площади квартиры по ключу, если новое значение меньше старого.
